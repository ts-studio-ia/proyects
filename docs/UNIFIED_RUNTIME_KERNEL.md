# Unified Runtime Execution Kernel (Fase 26.7)

Kernel único de ejecución gobernada para unificar Cognitive, Policy, Constitutional, MCP, Reality Bridge, Controlled Apply, Ledger, Knowledge y Replay.

## Ruta obligatoria
Human Goal → Cognitive → Policy → Constitutional → MCP → Reality Bridge → Controlled Apply → Ledger → Knowledge Graph → Replay → Operator Summary.

## Principios
- deny-by-default si falta decisión crítica
- tenant/project scope obligatorio
- mutaciones con approval + controlled apply
- secretos por referencia Vault
- eventos críticos al ledger
- replay obligatorio
