# External Provider Model

Adapters externos modelados sin side effects en constructor y `enabled=false` por defecto.

## Proveedores modelados
GitHub, GitLab, Local FS, Docker, Kubernetes/K3s, Vercel, PostgreSQL/Drizzle, Redis, Milvus, MinIO/S3/R2, OpenTelemetry, NATS/Queue, Vault, Terraform, Tailscale, Kyverno, y proveedores MCP equivalentes.
